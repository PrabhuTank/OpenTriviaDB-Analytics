# Open Trivia Database Analytics
**dashboard/** Power BI dashboard (.pbix) file and accompanying screenshot.<br>
**database/** MySQL database configuration and schema definitions.<br>
**docs/** Documentation files and usage guides.<br>
**scripts/** Supporting scripts and utilities.<br>
**etl_pipeline.py** Main ETL pipeline script.<br>
**etl_pipeline_production.py** ETL pipeline script for production use.<br>
**OpenTriviaDB-Job.bat** Batch file which will be scheduled to run hourly, daily, weekly or monthly on Windows.<br>
